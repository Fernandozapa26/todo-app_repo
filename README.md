# TODO App – React + Vite

Aplicación desarrollada como parte del taller  
**GA1-220501096-03-AA1-EV10 – Manejo de Rutas y APIs REST en React**.

##  Tecnologías utilizadas
- React
- Vite
- React Router DOM
- Fetch API
- Node.js LTS

---

##  Funcionalidades implementadas

### Obligatorias
1. Navegación entre páginas con react-router-dom  
2. Página Todos con consumo de API (useEffect + fetch)  
3. Listado dinámico de todos desde API  
4. Formulario controlado para crear nuevos todos  
5. Marcar todos como completados/pendientes  
6. Eliminar todos  
7. Estados de carga (“Cargando…”)  
8. Manejo de errores con try/catch  
9. Refactorización en componentes

###  Reto (placeholders)
10. Editar todos (estructura inicial creada)  
11. Filtros de tareas (estructura inicial creada)
